## **random_string.c**

This function generate a random string.

```bash
$> cc -std=c99 -Wall random_string.c -o random_str
> ./random_str
Input the length of the string: 70
4D3@{Xq3165avLO&Cl&+vpeJiOPv;VS73$!6NejHsgHNRjpgKhl&MGCV]}{1JV0UL=S4a@
```
